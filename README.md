# RailGuard AI

## Problem Statement

Railway track defects such as cracks, missing bolts, and missing clips can lead to serious accidents and service disruptions.

## Solution

RailGuard AI is an AI-powered railway inspection system that automatically detects track defects using YOLOv8 and generates risk assessments in real time.

## Features

* Crack Detection
* Missing Bolt Detection
* Missing Clip Detection
* Risk Scoring Engine
* SQLite Database Logging
* Streamlit Dashboard
* Visual Detection Reports

## Tech Stack

* Python
* YOLOv8
* Streamlit
* SQLite
* Pandas

## How to Run

pip install -r requirements.txt

streamlit run app_py.py

## Dashboard

streamlit run dashboard.py
